## 📖 项目简介

定时壁纸是一款自动收集和展示高清壁纸的工具。本仓库为壁纸采集服务，每小时自动从多个优质壁纸源采集最新高清壁纸，让你随时拥有精美的桌面背景。

## ✨ 功能特性

- 🕐 **自动更新**: 每小时自动采集全网最新高清壁纸
- 🖼️ **高清画质**: 所有壁纸均为高清分辨率
- 🌐 **多源采集**: 支持 Pixabay、Wallhaven 等多个优质壁纸源
- 📱 **响应式设计**: 网页端自适应各种屏幕尺寸
- 🔄 **实时预览**: 支持在线预览和下载原图

- **操作系统**: Windows 10/11
- **网络**: 需要互联网连接以更新壁纸

## 🌐 在线预览

访问 [dw6.gmij.win](https://dw6.gmij.win) 在线浏览所有壁纸

## 🛠️ 技术实现

- 使用 GitHub Actions 定时任务自动采集壁纸
- 图片采用懒加载技术，提升页面加载速度
- 响应式瀑布流布局，适配各种设备

## 📝 开发者说明

生成新仓库后，需要做的几件事：

1. 修改 `main.yml` 的定时器配置
2. 在新项目中创建一个 `PAT` 的安全密钥，用于访问仓库
3. 在 `action.gmij.win` 中添加新的 action 脚本
